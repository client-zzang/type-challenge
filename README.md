# Type Challenge

[type-challenges](https://github.com/type-challenges/type-challenges)를 함께 풀며 Typescript 타입 시스템에 대한 이해를 높이는 스터디입니다.

## 규칙

1. **공통 문제 5개**를 쉬움 난이도부터 순서대로 도전합니다.
2. **목표는 개인의 Typescript 성장**입니다. AI 사용, 정답 참고 등 풀이 방법은 자율에 맡깁니다.
3. 한 주의 목표를 달성하지 못하면 패널티가 부여되며, 일정 횟수 이상 누적 시 **커피 쏘기**☕
4. 각자 폴더에 풀이를 작성하며, **답안과 함께 접근법이나 배운 점**을 자유롭게 기록합니다.
5. **매주 금요일**에 의미 있던 문제나 공유하고 싶은 풀이, 배운 점 등을 공유하는 시간을 갖습니다.
